n8n AI Workflows

Reusable n8n workflow templates for AI automation, news aggregation and WordPress publishing.

Overview

This repository provides reusable n8n workflow templates focused on AI-powered content automation.

The goal is to help creators, developers and small businesses automate:

- RSS news collection
- AI summarization
- WordPress publishing
- Social media content generation

Included Workflows

RSS to WordPress Template

Basic workflow structure for collecting RSS content and preparing WordPress drafts.

Gemini Summary Template

Template for generating article summaries using Gemini.

Threads Post Template

Template for creating social media posts from summarized content.
